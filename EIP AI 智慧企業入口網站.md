## 📋 系統概述

本系統為企業入口網站（Enterprise Information Portal），依據公司規劃架構設計，整合以下五大核心功能模組：

1. **首頁** - 萬年曆、最新消息、天氣、分機查詢、系統管理 (員工帳號維護、簽核層級同步)。
2. **員工專區** - 訂餐、福委會、意見箱、問卷、安全管理、變更密碼。
3. **行政專區** - 
   - **人力資源部**：公告與時程管理。
   - **總務部**：訂餐管理、會議室預約、辦公文具申領、商務名片印製。
   - **法務部**：合約作業準則與保密協定。
   - **資訊部**：資安服務與障礙申報。
   - **財務部**：出差核銷基準匯率、匯款手續費事宜。
4. **ERP 資訊** - 連結外部系統（產銷、RMA、照明工程專案）。
5. **知識管理** - 合約管理、教育訓練、環安衛政策、ISO 文件、條碼管。

## 🚀 本地啟動方式

### 前置需求
- **PHP 7.4+** 以上版本
- **MySQL 5.7+**（可選，未連線時自動切換為 Session 模擬模式）
- 建議使用 **XAMPP**、**Laragon** 或 **AppServ**

### 啟動步驟

**方法一：使用 PHP 內建伺服器（最快）**
```powershell
# 在專案目錄下執行
php -S localhost:8000
```
開啟瀏覽器，前往：`http://localhost:8000/login.php`

**方法二：使用 XAMPP / Laragon**
1. 將整個 `/eip` 資料夾放入 XAMPP 的 `htdocs` 目錄
2. 啟動 Apache 和 MySQL
3. 前往：`http://localhost/eip/login.php`

## ⚙️ Session Fallback 自動備援機制

`config.php` 內建智慧偵測機制：

```
MySQL 連線成功  →  使用正式 MySQL 資料庫
      ↓ 失敗
Session 模擬資料庫  →  完整功能照常運作（含表單提交、統計）
```

- 頁面頂端會顯示目前資料庫連線狀態（橘色警示 / 綠色正常）
- Session 模擬模式下，所有操作在當次瀏覽器 Session 中正常保存

## 📝 各頁面功能摘要

### 🏠 首頁 `index.php`
- **萬年曆**：JavaScript 動態渲染，可切換月份。
- **最新消息 & 電子月刊**：彈出式 `<dialog>` Modal 顯示。
- **分機查詢**：前端即時模糊查詢（分機/姓名/部門/Email）。

### 👔 員工專區 `employee/`
- **留言板、福委會、廠區安全、內控問卷**。
- **台北訂餐**：點擊超連結至台北訂餐網站（Nancy 訂餐）。
- **變更密碼**：支援 Bcrypt 雜湊加密變更。

### 🏢 行政專區 `admin/`
- **總務部 - 會議室預約**：防止時段衝突。
- **總務部 - 文具與名片申請**。
- **總務部 - 訂餐管理**：統計今日同仁的便當總量，以便總務發單給商家。
- **財務部 - 財務核銷**：提供出差基準匯率、跨行轉帳手續費政策與常用財務報銷表單下載。

### ⚙️ 系統管理 (後台)
- **員工帳號維護**：Director 權限可新增與註銷同仁帳號。
- **簽核層級管理**：模擬從人事系統同步員工最新的簽核權限（Director、Manager、Employee），並展示圖形化職級樹狀圖。

### 📄 知識管理 `contract/`
- **合約管理系統** (同一主機，獨立工作區 `*` 範例)。
- **教育訓練與環安衛政策**。

## 🔄 版面佈局重大調整紀錄 (2026-06-25)

本系統已於 2026-06-25 進行重大版面升級，將全站架構重構為符合使用者規劃之**「橫向頂部 Header (大選單) + 內容三欄式」**佈局，以提昇使用者體驗與介面質感。

### 1. 全站導覽與三欄式架構
* **頂部 Header (大選單)**：包含首頁、員工專區、行政專區、ERP 資訊與知識管理五大分類及使用者面板。
* **三欄式內容 Grid**（桌面端 3-6-3 分配，行動端小於 `992px` 時自動堆疊為直向單欄）：
  * **左欄 (Col-3)**：專區導覽子選單卡片。
  * **中欄 (Col-6)**：核心主要功能與資訊卡片。
  * **右欄 (Col-3)**：直向輔助小工具（日曆、天氣、快速按鈕等）。

### 2. 各頁面三欄式配置
* **首頁 (`index.php`)**：左欄為首頁導覽；中欄為最新消息與分機查詢；右欄為萬年曆、天氣及考勤同步。
* **員工專區 (`employee/index.php`)**：左欄為員工專區導覽；中欄為同仁留言板與內控問卷；右欄為福委園地公告與廠區安全守則。
* **行政專區 (`admin/index.php`)**：左欄為行政導覽與總務子項；中欄為四大行政部門（HR、法務、IT、財務）公告；右欄為總務快捷通道。
* **知識管理 (`contract/index.php`)**：左欄為知識導覽與外部連結；中欄為合約起草與現存清單；右欄為教育訓練與環安衛。
