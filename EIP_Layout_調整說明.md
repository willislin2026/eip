# EIP 橫向頂部大選單與三欄式佈局調整說明

本文件記錄了本次「英群 EIP 企業入口網站」在視覺與版面佈局上的重大重構。我們將全站改為**「橫向頂部 Header (大選單) + 內容三欄式」**的現代化 premium 排版，使各分頁的視覺美感與操作動線達到高度一致。

---

## 📐 重構前後佈局對照

| 項目 | 重構前配置 | 重構後最新配置 (本次調整) |
| :--- | :--- | :--- |
| **主架構 Layout** | 左側固定側邊欄 (Sidebar) | **橫向頂部導覽列 (Navbar)** + **下方內容區三欄式 Grid** |
| **主選單選單** | 位於左側手風琴摺疊選單 | 位於頂部橫向大選單（首頁、員工專區、行政專區、ERP 資訊、知識管理） |
| **左欄 (Col-3)** | 無 (或被側邊欄佔用) | **專區導覽與子選單卡片** (`sidebar-column`)，放置該專區的快速錨點或子功能連結。 |
| **中欄 (Col-6)** | 主要內容在單一區塊強行拉寬 | **主要功能與資訊卡片** (`main-column`)，放置核心的操作表單、主要列表等。 |
| **右欄 (Col-3)** | 無 | **直向輔助小工具與卡片** (`layout-right`)，放置行事曆、天氣、公告守則或快捷入口。 |
| **行動裝置端** | 漢堡按鈕與隱藏側邊欄 | navbar 自適應，小於 `992px` 時三欄自動堆疊為 `col-12` 直向排列。 |

## 🎨 全域三欄式 CSS 框架 (`style.css`)

我們在 `style.css` 中建立了標準的 `12格` Grid 佈局類別，確保在不同頁面引入時能保持完美的比例：

* **左欄 `.sidebar-column` (占 3 格)**：主要做為專區內部的子導覽與跳轉選單。
* **中欄 `.main-column` (占 6 格)**：主要放置核心的功能模組與即時資訊。
* **右欄 `.layout-right` (占 3 格)**：做為輔助工具、天氣、日曆或公告放置處。
* **響應式斷點 (Media Query)**：當視窗寬度小於 `992px` 時，強制將 `.sidebar-column`, `.main-column`, `.layout-right` 重設為 `grid-column: span 12 !important`，以實現流暢的直向堆疊。

## 📝 各專區三欄佈局配置摘要

### 🏠 1. 首頁 (`index.php`)
* **左欄**：首頁快速導覽（包含消息公告、分機查詢、系統小工具等快速錨點）。
* **中欄**：最新消息與電子月刊列表、同仁分機模糊查詢與結果。
* **右欄**：萬年曆小工具、今日天氣小工具、人事考勤同步快捷卡。

### 👔 2. 員工專區 (`employee/index.php`)
* **左欄**：員工專區導覽（包含台北訂餐、福委園地、人資申請、意見箱、留言板、廠區安全、問卷、變更密碼等錨點與連結）。
* **中欄**：同仁線上留言板（留言提交表單與即時消息列表）、內控問卷調查表單（q1, q2, q3 選擇題）。
* **右欄**：福委園地公告（國內旅遊補助申請與發放時程）、廠區安全守則（消防逃生與避難規範）。

### 🏢 3. 行政專區 (`admin/index.php`)
* **左欄**：行政專區導覽（人力資源部、法務部、資訊部、財務部、總務部訂餐、會議室預約、文具名片申領入口）。
* **中欄**：人力資源部公告、法務部公告（NDA作業）、資訊部守則（密碼要求）、財務部出差專區。
* **右欄**：總務快捷小工具（訂餐管理、會議室預約、文具申領、名片印製快捷按鈕）。

### 📄 4. 知識管理 (`contract/index.php`)
* **左欄**：知識管理導覽（起草合約、合約清單、教育訓練、環安衛政策，以及外部 ISO 文件、條碼系統快速連結）。
* **中欄**：起草與登錄新合約表單、限制閱讀合約現存清單。
* **右欄**：教育訓練課程推薦列表、環安衛 (EHS) 政策卡片。
